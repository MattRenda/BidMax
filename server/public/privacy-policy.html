Privacy Policy

Last updated: March 2026
Short version: BidMax does not collect, store, or share any personal information about you. Ever.

What is BidMax?

BidMax is a Chrome extension that analyzes auction lots on BidRL.com and displays a calculated maximum bid and estimated resale profit directly on the auction page. It is designed for resellers who want to make faster, more informed bidding decisions.

Data We Do Not Collect

BidMax does not collect, transmit, or store any of the following:

  • Personal identifiable information (name, email, address)
  • Financial or payment information
  • Authentication credentials
  • Browsing history or web activity
  • Location data
  • User activity (clicks, keystrokes, mouse movements)
  • Any data about you as a person

What Happens When You Use BidMax

When you browse an auction page on BidRL.com, BidMax reads the lot title and bid information visible on that page and sends it to a private analysis server to estimate resale value. Specifically:

  • Lot title and bid amount — sent to the BidMax analysis server to generate a max bid estimate. No personal information is included.
  • Your settings (target margin, buyer's premium, etc.) — stored locally in your browser using Chrome's built-in storage. Never transmitted to any server.
  • Analysis results — cached locally in your browser for up to 2 hours to avoid redundant API calls. Never shared with third parties.

Third-Party Services

BidMax uses the Anthropic API to generate lot value estimates. Lot title text is sent to Anthropic's API for this purpose. Anthropic's privacy policy can be found at anthropic.com/privacy. No personal user data is included in these requests.

Data Sharing

BidMax does not sell, rent, or share any data with third parties. The only external communication is sending auction lot titles to the analysis server for the purpose of generating bid estimates — which is the core function of the extension.

Children's Privacy

BidMax is not directed at children under the age of 13 and does not knowingly collect any information from children.

Changes to This Policy

If this privacy policy changes, the updated date at the top of this page will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
